download postGreSql from https://www.postgresql.org/download/

install the exe file, while asking for password, set to "random"

download pgadmin from https://www.pgadmin.org/

install the exe file

open pgadmin, when asking for password, set to "random"

while connected to db with pgadmin, create a new database named "quizoclock"

run in the console, "npm install" from the root directory

run in the console, "node initdb.js" from the directory "./data"

run in the console, "node --watch index.js" from the root directory
